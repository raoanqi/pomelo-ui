# Button 按钮

按钮可以用于响应用户的点击操作。

## 基本使用

使用`type`属性来指定按钮的类型。  
`type`属性支持四种值：`primary`（默认）、`secondary`，`text`，`outline`。

:::demo

```vue

<template>
  <p-button type="primary">Primary</p-button>
  <p-button type="secondary">Secondary</p-button>
  <p-button type="text">Text</p-button>
  <p-button type="outline">Outline</p-button>
</template>

```

:::
